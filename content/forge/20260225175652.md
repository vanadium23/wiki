Здесь всё чаще иду в сторону [[20260225170023|ванильное использование]]. Потому, что раньше было супер кастомное решение с кучей dataview, а теперь вроде как и на встроенных bases можно вполне себе неплохо жить.

### TaskNotes

Плагин, который реализуется поверх новых bases. Делает по одной заметке на задачу.

Что настраивают до себя (актуально для версии v4.4.0 на 2026-02-25):
1. task properties: расширяю статусы тремя: blocked, review, cancelled
2. task properties: для проектов делаю фильтр по папке actions/
3. task properties: title в property, filename zettelkasten format 
4. apperance & ui: checklist progress для отображения прохождения чеклиста
5. appearance and ui:  click - open note

### LiveSync

Это self hosted решение поверх couchdb, чтобы можно было синхронизировать заметки. У неё беда с первоначальной настройкой, потому что надо прогнать два скрипта:
1. https://raw.githubusercontent.com/vrtmrz/obsidian-livesync/main/utils/couchdb/couchdb-init.sh
    1. этот нужен, чтобы проставить CORS и настройки для базы
2. https://raw.githubusercontent.com/vrtmrz/obsidian-livesync/main/utils/flyio/generate_setupuri.ts
    1. этот нужен, чтобы получить URL для начального setup'а
