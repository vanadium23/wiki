### Ревизия 2024-11-25

ChernoWiki построен с помощью [quartz](https://quartz.jzhao.xyz/), который я пытаюсь [[./кастомизация движка Quartz|настроить под себя]]. Сами заметки хранятся в [[obsidian|obsidian]], в котором настроен плагин [[obsidian-digital-garden|obsidian-digital-garden]]. Для того, чтобы заметка была опубликована, у неё в [[frontmatter|frontmatter]] должно быть прописано свойство:
```yaml
dg-publish: true
```

Всё более менее стандартно:
- код лежит в [github репозитории](https://github.com/vanadium23/wiki/)
- собирается через [github actions](https://github.com/vanadium23/wiki/blob/master/.github/workflows/github-pages.yaml)
- распологается на github pages
- закрыто всё Cloudflare
